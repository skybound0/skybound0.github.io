Designed and built a thrust vectoring VTOL vehicle around a 90mm EDF running the SingleCopter ArduPilot configuration.

It flies, kinda. ArduPilot seems to handle the gyroscopic procession well enough, but I'm still working on tuning the PID and many other aspects of the software.

All 3D printed parts were designed to be as easy to print as possible with minimal supports.

This project was inspired by the [Ikarus](https://www.youtube.com/watch?v=RMeEh5OUaDs/) project.
